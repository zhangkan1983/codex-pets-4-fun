# Zoro Codex Pet

An unofficial, fan-made custom pet for the Codex desktop app, assembled from supplied Zoro pose materials.

![Preview](assets/preview.png)

## Install

1. Download `assets/spritesheet.webp` and `pet.json`.
2. Create the directory `~/.codex/pets/zoro-santoryu/`.
3. Put both downloaded files in that directory, keeping the filename `spritesheet.webp`.
4. In Codex, open **Settings > Pets**, choose **Refresh**, then select **Zoro**.

## Package

- `assets/spritesheet.webp` — transparent 1536 x 1872 WebP animation atlas.
- `pet.json` — Codex pet metadata pointing to `spritesheet.webp`.
- `assets/preview.png` — contact sheet of the nine animation states.
